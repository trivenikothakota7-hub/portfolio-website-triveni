Frontend Devlop

I built a responsive and dynamic user interface using HTML, CSS, React.js, and Tailwind CSS. React.js powered interactive components, while Tailwind CSS enabled rapid styling with a utility-first approach. The website adapts to different screen sizes, ensuring an optimal experience on both desktop and mobile

Backend

The backend was developed using Node.js and Express.js to create a fast and scalable server. Express.js handled API routes for user authentication and data management, ensuring efficient communication between the frontend and backend.

Database

I used MongoDB ,SQL as the database, with Mongoose for easy schema definition and CRUD operations. The database stores user and application data, with seamless integration into the backend to handle requests and provide data to the frontend.

Authentication and Authorization

Authorization is handled by checking the user's roles and permissions after successful authentication. Different routes and actions are accessible only if the user has the necessary permissions or roles (e.g., admin or user), ensuring restricted access to certain parts of the application based on user privileges
